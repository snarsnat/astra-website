GitHub OAuth Test - Vercel Deployment

Test GitHub OAuth Flow

Ready to test GitHub OAuth integration...

API Endpoints:

  • /api/auth/github - Start OAuth flow
  • /api/auth/github/callback - Handle callback from GitHub

Environment Variables Needed:

  • GITHUB_CLIENT_ID - Your GitHub OAuth App Client ID
  • GITHUB_CLIENT_SECRET - Your GitHub OAuth App Client Secret
  • GITHUB_CALLBACK_URL - Callback URL (default: https://astra-website-vercel-test.vercel.app/api/auth/github/callback)

Steps to Configure:

  1. Create GitHub OAuth App at GitHub Developer Settings
  2. Set Authorization callback URL to: https://astra-website-vercel-test.vercel.app/api/auth/github/callback
  3. Add environment variables in Vercel project settings
  4. Deploy to Vercel
